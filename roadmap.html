Release Roadmap

A realistic milestone plan that keeps scope under control while proving core gameplay.

MVP (Minimum Viable Product)

  • One run start → one room/wave → rewards → return
  • Inventory + hotbar equip stable
  • 1 enemy type + basic loot table

Vertical Slice

  • 2–3 rooms or waves
  • 2 enemy types + one “mini-boss” pattern
  • Rarity tiers + upgrade path

Polish

  • Hit feedback + readable VFX
  • UI clarity improvements
  • Performance + cleanup passes

Release

  • Onboarding + tutorial hints
  • Balance pass
  • Launch checklist + bug triage
