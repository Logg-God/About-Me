Generative AI Workflow

I use AI as a productivity tool for scaffolding modules, generating test cases, summarizing bugs, and refactoring for readability. I validate all changes through testing, logging, and code review.

What I use AI for

  • Service/module scaffolding and folder structure
  • Test checklists and edge cases
  • Refactoring scripts into smaller, readable functions
  • Turning design goals into implementable tasks

How I validate

  • In-game testing in isolated test maps
  • Server-side logging for damage and reward events
  • Code review from mentors (including my dad)
