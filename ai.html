AI Workflow

Generative AI Workflow

AI as a tool: scaffolding, refactoring, test ideas — with validation and review.

How AI helps

I use AI to accelerate development while keeping ownership of the design and logic. AI is most valuable for turning ideas into structured tasks and reducing time spent on boilerplate.

  • Scaffold module/service templates and folder structure
  • Generate test checklists + edge cases
  • Refactor scripts into smaller, readable functions
  • Summarize bugs and propose fixes

How I validate

All AI-assisted changes are validated through testing and review before they’re considered done.

  • In-game testing in isolated test maps
  • Server-side logging for damage and reward events
  • Code review from mentors (including my dad)
  • Small commits with clear messages to track changes

Guiding principle

AI speeds up implementation, but I stay responsible for correctness, security (anti-exploit), code clarity, and game feel. If something is confusing, it gets rewritten.
