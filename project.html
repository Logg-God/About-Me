Project Overview

Elevator pitch: Dungeon Fighter is a floor-based PvE dungeon game where players fight through rooms, defeat a boss, and progress to the next level.

Core gameplay loop

  • Spawn → Explore → Fight enemies → Earn loot/coins → Boss fight → Next floor

MVP (Vertical Slice)

The MVP is the smallest complete version that proves the core loop is fun.

  • 1 floor (start → combat → boss → reward)
  • 1 weapon: sword
  • 2 enemies: melee chaser + tank bruiser
  • 1 boss: 2 attacks + simple phase change
  • UI: health bar + objective
  • Rewards: coins or basic loot drop
  • Fail state: death → restart
